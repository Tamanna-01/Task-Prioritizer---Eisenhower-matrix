# 1. Introduction

FocusGrid is a simple and interactive task management application that helps users organize their work based on priority. Instead of maintaining a long list of tasks, FocusGrid allows users to visually arrange tasks according to their urgency and importance using the Eisenhower Matrix.

The application has been designed with simplicity in mind, allowing users with little or no technical knowledge to use it effectively.

---

# 2. Purpose of the Application

Many people create long to-do lists but struggle to decide what should be completed first. FocusGrid helps solve this problem by encouraging users to categorize tasks based on their importance and urgency.

The application aims to:

- Reduce confusion while planning daily work.
- Improve productivity.
- Help users focus on important tasks.
- Minimize distractions.
- Provide a visual overview of work priorities.

---

# 3. Understanding the Eisenhower Matrix

The application divides tasks into four categories.

| Quadrant                   | Meaning                                      | Recommended Action |
| -------------------------- | -------------------------------------------- | ------------------ |
| Urgent & Important         | Critical tasks requiring immediate attention | Do First           |
| Not Urgent & Important     | Long-term goals and planning                 | Schedule           |
| Urgent & Not Important     | Tasks that can be delegated                  | Delegate           |
| Not Urgent & Not Important | Low-value or distracting tasks               | Eliminate          |

This method helps users focus on meaningful work instead of simply reacting to urgent requests.

# 4. System Overview

The application consists of three primary sections.

### Quick Add

This section allows users to quickly create new tasks.

---

### Inbox

All newly created tasks appear here before they are categorized.

Users can:

- Review tasks
- Edit task descriptions
- Delete unwanted tasks
- Drag tasks into the appropriate quadrant

---

### Focus Matrix

The Focus Matrix contains the four priority quadrants where tasks are organized based on their urgency and importance.

# 5. Getting Started

Using FocusGrid requires only a few simple steps.

### Step 1

Open the application in your web browser.

---

### Step 2

Locate the **Quick Add** section.

---

### Step 3

Type your task into the input box.

Example:

> Complete project presentation

---

### Step 4

Click the **Add Task** button.

The task will immediately appear inside the Inbox.

# 6. Features

Users can quickly create new tasks using the Quick Add section.

If a mistake is made, users can click the edit option and modify the task description.

Tasks that are no longer required can be removed from the application.

# 7. Step-by-Step Workflow

## Step 1: Create a Task

Type a task into the input field.

Example:

> Prepare for client meeting

Click **Add Task**.

↓

Task appears inside the Inbox.

---

## Step 2: Review the Task

Read the task and decide:

- Is it important?
- Is it urgent?

---

## Step 3: Drag the Task

Drag the task from the Inbox.

Move it to the appropriate quadrant.

---

## Step 4: Continue Organizing

Repeat the same process for all remaining tasks.

Eventually, the Inbox becomes empty while the Focus Matrix contains all categorized tasks.

---

## Step 5: Edit if Required

At any time, users may:

- Edit task text
- Delete tasks
- Move tasks between quadrants

---

## Step 6: Download

Once satisfied with the organization:

Click:

**Download Board**

The application automatically saves the matrix as a high-quality PNG image.

# 9. Downloading Your Board

To save your work:

1. Complete the organization of your tasks.
2. Click **Download Board**.
3. Wait a few seconds.
4. The image is automatically downloaded.

### Note

If the board contains no tasks, the application displays an error message instead of generating an empty image.

---

# 10. Tips for Effective Usage

To get the most benefit from FocusGrid:

- Add every task that comes to mind before organizing.
- Focus on completing the "Urgent & Important" quadrant first.
- Regularly review the "Not Urgent & Important" section, as these tasks often contribute most to long-term success.
- Remove unnecessary tasks from the "Not Urgent & Not Important" quadrant to reduce distractions.
- Download your board at the end of each planning session for future reference.

---

# 11. Frequently Asked Questions (FAQs)

### Q1. Can I edit a task after creating it?

Yes. Every task can be edited at any time.

---

### Q2. Can I move a task to another quadrant?

Yes. Simply drag and drop the task into the desired quadrant.

---

### Q3. Can I delete a task?

Yes. Tasks can be removed whenever they are no longer needed.

---

### Q4. Does the application require an internet connection?

The application runs entirely in the browser after it has loaded. Depending on how it is deployed, internet access may only be needed to initially open the application.

---

### Q5. Why can't I download an empty board?

The application prevents downloading when no tasks are present to avoid generating meaningless files.

---

# 12. Troubleshooting

| Issue                   | Possible Solution                                          |
| ----------------------- | ---------------------------------------------------------- |
| Unable to create a task | Ensure the input field is not empty.                       |
| Task not moving         | Drag and drop the task directly into a quadrant.           |
| Download not working    | Verify that at least one task has been added to the board. |
| Task missing            | Check whether it has been moved to another quadrant.       |
| Button not responding   | Refresh the page and try again.                            |

# 13. Conclusion

FocusGrid simplifies task management by transforming traditional to-do lists into a visual prioritization system. Through its intuitive interface and drag-and-drop interaction, users can quickly identify what requires immediate attention, what should be planned for later, what can be delegated, and what can be eliminated.

The application is designed to be easy to learn, making it suitable for users of all technical backgrounds while promoting more effective decision-making and productivity.
