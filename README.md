# 🎯 FocusGrid

> A modern, responsive task prioritization application based on the **Eisenhower Matrix**, built with **React, TypeScript, and Vite**.

FocusGrid helps users organize their daily tasks by categorizing them into four priority quadrants based on **Urgency** and **Importance**, enabling better decision-making and improved productivity.

---

## ✨ Features

- 📝 Create new tasks instantly
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- 🎯 Organize tasks using the Eisenhower Matrix
- 🖱️ Smooth drag-and-drop interactions
- 📥 Download the board as a high-quality PNG
- 🔔 Toast notifications for user feedback
- 🌙 Modern Apple/Linear-inspired dark UI
- 📱 Fully responsive design

## 🎮 How to Use

### 1. Add Tasks

Use the **Quick Add** section to create new tasks.

### 2. Organize Tasks

New tasks appear in the **Inbox**.

Drag each task into one of the four quadrants:

- 🔴 Urgent & Important → **Do First**
- 🔵 Not Urgent & Important → **Schedule**
- 🟠 Urgent & Not Important → **Delegate**
- ⚪ Not Urgent & Not Important → **Eliminate**

### 3. Manage Tasks

You can:

- Edit tasks
- Delete tasks
- Move tasks between quadrants

### 4. Export

Click **Download Board** to save your completed matrix as a PNG image.

## 🔍 Why the Eisenhower Matrix?

The Eisenhower Matrix helps users prioritize tasks by answering two simple questions:

- Is it **Urgent**?
- Is it **Important**?

| Quadrant | Action |
|----------|--------|
| Urgent & Important | Do First |
| Not Urgent & Important | Schedule |
| Urgent & Not Important | Delegate |
| Not Urgent & Not Important | Eliminate |

This method promotes better decision-making rather than simply creating longer to-do lists.
